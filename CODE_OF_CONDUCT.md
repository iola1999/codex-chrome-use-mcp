# Code of Conduct

This project expects practical, respectful collaboration.

## Expected Behavior

- Be clear, factual, and constructive.
- Discuss technical disagreements with evidence.
- Respect user privacy and avoid requesting sensitive browser profile data.
- Keep issues and pull requests focused on the project.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private user data, secrets, cookies, tokens, or session files.
- Intentionally misleading reports or reproduction steps.
- Spam or unrelated promotion.

## Enforcement

Maintainers may edit, hide, or remove comments and may block contributors who repeatedly violate these expectations.

For private reports, email `iola1999@foxmail.com`.
